# 🛒 COMIF Wrapped

**COMIF Wrapped** is a data-driven project inspired by Spotify Wrapped, designed to analyze and visualize consumption patterns within a school grocery store and pub.

It transforms raw transaction data into personalized insights, visual summaries, and shareable reports.

---

## 🚀 Concept

The project takes consumption data (Excel files) and automatically generates:

- 📊 Data visualizations  
- 🧾 Personalized summaries  
- 📄 A clean PDF report (Wrapped-style)  
- 📧 Automated email delivery to users  

---

## ⚙️ Features

### 📥 Data Processing
- Reads and processes raw Excel transaction data  
- Cleans and structures consumption information  

### 📊 Visualization & Insights
- Generates key metrics (top products, frequency, spending patterns)  
- Creates dynamic plots and charts  

### 🎨 Template Rendering
- Injects results into a predefined visual template  
- Produces a “Spotify Wrapped”-style layout  

### 📄 PDF Generation
- Automatically exports reports as PDF files  
 
 
---

## 🛠️ Tech Stack

- Python  
- Pandas (data processing)  
- Matplotlib / Seaborn / Plotly (visualization)  
- PDF generation tools  

---

## 📈 Output

Each user receives a personalized report including:

- Top consumed products 🥇  
- Spending insights 💸  
- Consumption habits 📊  
- Visual summaries
